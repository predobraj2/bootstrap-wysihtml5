h1. Overview

WYSIWYG editors suck. Fortunately Xing are rectifying it by building "wysihtml5":https://github.com/xing/wysihtml5. Styling and set up is a little lacking though so I thought I'd build a "twitter bootstrap":http://twitter.github.com/bootstrap/ theme for it.

h1. Examples

* "http://jhollingworth.github.com/bootstrap-wysihtml5":http://jhollingworth.github.com/bootstrap-wysihtml5/

h1. Usage

Download:

* "bootstrap-wysihtml5.js":https://raw.github.com/jhollingworth/bootstrap-wysihtml5/master/src/bootstrap-wysihtml5.js
* "bootstrap-wysihtml5.css":https://raw.github.com/jhollingworth/bootstrap-wysihtml5/master/src/bootstrap-wysihtml5.css
* "wysihtml5":https://github.com/xing/wysihtml5
* "Twitter Bootstrap":http://twitter.github.com/bootstrap/

<pre>
	<textarea id="some-textarea" placeholder="Enter text ..."></textarea>
	
	<script type="text/javascript">
		$('#some-textarea').wysihtml5();
	</script>
</pre>
